## [0.1.0] - 2026-05-18
### Added
- Initialized The Myelin Engine project structure.
- Developed Neural Firehose simulator (1,024 channels at 20kHz).
- Implemented C++ Bit-Stream Delta Encoder (BSDE) with sub-1ms latency.
- Implemented Adaptive Spatiotemporal Squelch (ASS) in C++ using explicit ARM NEON 128-bit vector intrinsics for zero-copy memory operations.
- Developed Quantum Entanglement Routing (QER) Telemetry Bridge prototype.
- Achieved total combined latency of 0.931ms with 37.8% bandwidth savings for Neuralink-tier performance.
